== Description ==

<strong>WP Block Pack</strong> is a pack of clean, useful and, customizable blocks for the WordPress block editor (Gutenberg). The blocks look great out of the box, and will make content writing a buzz. These blocks also responsive and will look great even in small screens.


## We brings You these awesome blocks

In additon to your Block Editor, we packed these 11 powerful blocks. Of course we are not feeling enough with these blocks, we always developing new blocks, new features while maintaining the existing one.

* Section
* Drop Cap
* Counter

* Buttons Pro
* Accordion
* Tabs
* Notice

* Advertisement
* Blockquote


== Frequently Asked Questions ==

= Why WordPress Block Editor? =

The new content editor that came out in WordPress 5 gives you a more visual way of creating your content.

With it, you write post content and pages using blocks, and WP Block Pack plugin bring you a collection of cool blocks you gonna love.

= What is block? =

Block are the new shortcodes in WordPress 5.0.

These are elements that you add into your content to build your pages, like buttons, cards, videos, etc.

= Can I use WP Block Pack with other block plugins? =

Yes you can! WP Block Pack blocks could be use together with other blocks.

= Can I use WP Block Pack on client websites? =

Sure! You can use WP Block Pack on your client’s websites as well as yours.

= Will WP Block Pack slow down my website? =

Absolutely No! WP Block Pack is built with speed load in mind. Its designed and coded well to keep it lightweight and fast as possible.
